Place institution logo files for the "My path so far" slide in this folder.

Supported extensions:
- `.pdf`
- `.png`
- `.jpg`
- `.jpeg`

Expected slugs:
- `ufrj`
- `impa`
- `bonn`
- `eth`
- `epfl`
- `ulisbon`

Examples:
- `logos/ufrj.png`
- `logos/impa.pdf`
- `logos/eth.png`

If a logo file is missing, the slide shows a small placeholder box instead, so the presentation still compiles.
